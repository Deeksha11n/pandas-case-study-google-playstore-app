This project focuses on performing Exploratory Data Analysis (EDA) on a Google Play Store dataset using Python ( NumPy).
🔍 Analysis Tasks Performed
1. Basic Data Exploration
Display top 5 rows of the dataset
Display last 3 rows
Find dataset shape (rows & columns)
Get dataset information (datatypes, memory usage)
Generate overall statistics

3. App-Specific Insights
Count apps with titles containing "Astrology"
Calculate average app rating
Count total unique categories
Identify category with highest average rating

5. Ratings & Reviews Analysis
Count apps with 5-star rating
Calculate average number of reviews
Find app with maximum reviews
Display top 5 apps with highest reviews

7. Pricing Analysis
Count total number of Free vs Paid apps
Compare average rating of Free and Paid apps

9. Installs Analysis
Display top 5 apps with maximum installs
📊 Key Insights
Most apps belong to a few dominant categories
Free apps are significantly higher in number than paid apps
Certain categories consistently achieve higher ratings
Apps with higher installs tend to have more reviews
